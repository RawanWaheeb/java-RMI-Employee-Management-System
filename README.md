# Employee Management System (RMI + JDBC)

Distributed Java application using **Java RMI** for remote calls and **JDBC** for **MySQL** database persistence.

##  Requirements
- XAMPP (Apache & MySQL)
- Java JDK
- Maven

##  How to Run
1. Start MySQL in XAMPP.
2. Run `Server.java` (Wait for "Server is running" message).
3. Run `Client.java` to perform CRUD operations.
